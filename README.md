# WeWe RSS Template for Zeabur

更优雅的微信公众号订阅方式，用于在 [Zeabur](https://zeabur.com/) 快速部署 WeWe RSS 服务。

## 快速开始

1. 克隆或 Fork 本仓库
2. 登录 Zeabur 控制台
3. 选择「Template」→「Create Template」→「从 GitHub 导入」
4. 填写变量，如数据库信息和域名
5. 部署成功后，访问你的绑定域名并输入 AUTH_CODE 开始使用
